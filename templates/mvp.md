# MVP Definition

## MVP Objective
- Validate one painful workflow with one sellable feature.

## In Scope
- Core generation engine
- Single-page tool UX
- Fixed output schema
- Free limit + upgrade path

## Out of Scope
- Multi-feature dashboard
- Deep analytics
- Complex team workflows

## Success Metrics
- Activation: <metric and target>
- Engagement: <metric and target>
- Quality: <metric and target>

## Delivery Plan
- Week 1: core implementation + landing
- Week 2: SEO pages + conversion tuning
- Week 3: iterate from usage signals

## Launch Gate
- [ ] Core user flow works in <30 seconds
- [ ] Output quality is usable without edits
- [ ] Error handling covers invalid input and provider failures
- [ ] Payment/upgrade path is visible when limit hits
