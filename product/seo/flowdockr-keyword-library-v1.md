# Flowdockr SEO Keyword Library v1

Generated: 2026-03-04

## Scope

- Language: English-first
- Total keywords: 300
- Goal: capture high-intent freelancer problems and route traffic to Flowdockr tools

## Cluster Distribution

- negotiation-discount: 70
- pricing-strategy: 70
- client-communication: 60
- proposal-and-scope: 50
- rate-increase: 50

## Data Columns

- id: stable keyword id for automation
- keyword: target SEO query
- cluster: topical cluster
- intent: user job-to-be-done category
- target_route: recommended Flowdockr CTA destination
- content_type: guide / template / tool
- conversion_score: heuristic 1-5 (5 = strongest signup intent)
- competition_guess: heuristic low/medium based on long-tail specificity
- priority: P1/P2/P3 for build order

## First 100 Build Queue Recommendation

- P1 first within each cluster, then P2
- Keep 70% pages as template-heavy intent (template/script/email/calculator terms)
- Include one direct CTA module link above the fold and one in-content CTA

## KPI Loop (90 Days)

- Index coverage: indexed URLs / submitted URLs
- CTR: Search Console query CTR by cluster
- Activation CTR: guide page click-through to target_route
- Signup rate: sessions from SEO pages -> signup
- Qualified conversion: signup -> first successful generation

## Known Gaps

- Search volume and keyword difficulty are heuristic in v1; replace with API data (GSC, Ahrefs, Semrush, or Keyword Planner) in v2
- SERP feature overlap is not yet scored (forum-heavy vs SaaS-heavy SERP)
- No locale variants included yet to avoid multilingual index quality issues
