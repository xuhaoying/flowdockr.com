# FlowDockr Context

## Product Positioning
- 产品一句话：FlowDockr 是一个 Freelancer Deal Decision System，把客户消息转成「可执行谈判决策包」。
- 目标用户：自由职业者 / 独立开发者 / 设计师 / 咨询顾问（高频遇到压价与谈判）。
- 核心承诺：30 秒生成「立即可发回复 + 策略解释 + 价格方案 + 风险 + 下一步」。

## Non-Goals (MVP)
- 不做聊天机器人。
- 不做长篇课程。
- 不做 CRM（MVP 阶段不做）。

## MVP Success Metrics
- 用户愿意复制 `instant_reply`。
- 用户愿意截图分享输出卡片。
- 10 个测试用户里至少 2 个愿意付费（`>= 20%`）。
