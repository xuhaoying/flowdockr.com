- Keyword intent: high-intent, problem-first, action-oriented.

- pricing-strategy

## Landing + Tool SEO Focus

## Content Template (Guide)
1. Situation summary
2. Common mistakes
3. Correct strategy
4. Copy-paste example reply
5. CTA to FlowDockr tool

## Indexing Gate (Required)
- Dev/preview: `noindex,nofollow` + `robots disallow`.
- Launch-ready only: open indexing + submit sitemap.

## Launch Criteria for Indexing On
- Output quality stable across core use cases.
- No thin/duplicate autogenerated pages.
- At least one useful, non-generic answer per target query cluster.
