# Flowdockr Brand Pack

## Core direction
Flowdockr is positioned as a calm, trustworthy negotiation copilot for freelancers.
The visual system should feel:
- precise
- modern
- clean
- credible
- not gimmicky

## Logo concept
The icon uses two speech bubbles plus a bidirectional arrow.
Meaning:
- dialogue
- negotiation
- movement toward agreement
- structured back-and-forth instead of emotional chaos

## Included files
- `flowdockr-logo-primary.svg` — main wordmark logo
- `flowdockr-logo-primary.png` — PNG wordmark for quick use
- `flowdockr-logo-mark.svg` — standalone icon mark
- `flowdockr-favicon.svg` — favicon source
- `flowdockr-app-icon.png` — 1024x1024 app/icon tile
- `flowdockr-app-icon-light.png` — light-background app icon
- `flowdockr-og-image.png` — 1200x630 social sharing image

## Color palette
- Primary Indigo: `#4F46E5`
- Primary Dark: `#4338CA`
- Text: `#111827`
- Muted Text: `#6B7280`
- Background: `#FFFFFF`
- Soft Indigo Background: `#EEF2FF`

## Usage rules
1. Default to the full logo on landing pages and payment pages.
2. Use the icon mark for favicon, navbar, and compact mobile areas.
3. Keep plenty of whitespace around the mark.
4. Avoid shadows, neon gradients, and cyberpunk styling.
5. Prefer white or very light neutral backgrounds.

## Suggested product voice
- calm
- professional
- direct
- confidence without aggression

## Suggested hero copy
Flowdockr helps freelancers respond to lowball offers with structured, professional replies.
